# 静默团队 (SilentFlow)

> 🤝 智能团队协作管理系统  
> 通过**团队池架构**和**项目模板驱动**，实现高效、结构化的团队协作

[![Flutter](https://img.shields.io/badge/Flutter-3.8.1-blue.svg)](https://flutter.dev/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## 📖 项目简介

**静默团队**是一个基于Flutter开发的团队协作管理系统，采用**团队池架构**和**项目模板系统**，为团队提供结构化的协作环境。系统支持队长-队员角色管理、智能任务分配、项目模板创建，特别适合软件开发、学术研究、商业提案等协作项目。

### ✨ 核心特性

🎯 **团队池架构**
- 队长-队员角色权限体系
- 团队邀请码和成员管理
- 基于权限的功能控制

� **项目模板系统**  
- **8个专业模板**：软件开发、协作写作、学术论文、商业提案、技术文档、研究项目、营销活动、设计项目
- 结构化项目创建流程
- 预设任务步骤和里程碑

🧩 **智能任务管理**
- 任务模板驱动的子任务创建
- 基于成员技能的智能分配
- 任务依赖关系和进度跟踪

📊 **团队协作指数**
- 团队协作配合程度评估
- 成员贡献值量化分析
- 项目完成质量统计

## 🏗️ 技术架构

### 核心模型
```
TeamPool          - 团队池核心模型（队长管理、成员角色、权限系统）
TaskTemplate      - 任务模板系统（8个预定义模板 + 自定义）
MemberRole        - 成员角色枚举（队长/队员权限定义）
```

### 服务层
```
TeamPoolService   - 团队池业务逻辑
TaskTemplateService - 模板管理服务
TeamPoolProvider  - 状态管理Provider
```

### 界面架构
```
HomeScreen        - 团队协作指数、快速创建团队
TeamPoolScreen    - 我的团队、发现团队、团队统计
ProfileScreen     - 个人资料、快速创建项目模板
TaskBoardScreen   - 任务管理和分配
```

## 💡 项目模板

### 🖥️ 软件开发项目
**流程**：需求分析 → 系统设计 → 编码实现 → 测试验证 → 部署上线  
**适合**：3-6人团队，10-15天  
**技能**：编程、软件设计、测试

### ✍️ 协作写作项目
**流程**：大纲规划 → 内容撰写 → 同行评议 → 编辑修订 → 终审发布  
**适合**：2-5人团队，5-8天  
**技能**：写作、编辑、协作

### 🎓 学术论文协作
**流程**：文献综述 → 研究设计 → 数据收集 → 数据分析 → 论文撰写  
**适合**：2-4人团队，15-20天  
**技能**：学术写作、研究方法、数据分析

### 💼 商业提案协作
**流程**：市场调研 → 商业模式设计 → 财务规划 → 风险评估 → 提案撰写  
**适合**：3-6人团队，8-12天  
**技能**：商业分析、财务规划、演示制作

### 📋 技术文档协作
**流程**：需求文档 → 架构文档 → API文档 → 用户手册 → 文档审核  
**适合**：3-5人团队，10-15天  
**技能**：技术写作、文档规范、工具使用

## 🎨 界面展示

### 首页
- **团队协作指数**：周/月/总分统计展示
- **今日关键节点**：团队成员任务进展通知
- **快速创建团队**：项目模板选择对话框

### 团队池页面
- **我的团队**：参与的活跃团队项目
- **发现团队**：可加入的公开团队
- **团队统计**：协作数据和成就展示

### 个人资料
- **创建合作项目**：8个项目模板快速选择
- **我的团队**：直接跳转团队管理界面
- **统计展示**：完成任务数、团队参与数、贡献值、协作指数

## 🔧 使用指南

### 创建团队项目
1. 在首页点击"创建团队"或个人资料"创建合作项目"
2. 选择项目模板（软件开发、协作写作、学术论文等）
3. 填写团队信息和邀请成员
4. 系统自动生成结构化任务和里程碑

### 任务管理
1. 队长可基于成员技能智能分配任务
2. 队员可根据个人特长主动认领任务
3. 支持任务依赖关系和进度跟踪
4. 实时更新团队协作指数
